## Overview

This repository contains the complete theoretical, numerical, and experimental framework for **Null-Vector Gravity (NVG)** and its underlying dense matter model, the **Vacuum Mass Fraction (VMF)**.

The framework consists of three main pillars:

The model successfully resolves causality ($c_s \leq c$) and stiffness problems, yielding a maximum neutron star mass of $M_{\max} \approx 2.3\,M_\odot$. For a $10\,M_\odot$ black hole, this gives two horizons (inner Cauchy: 1.15 km, outer: 29.34 km) and a finite Kretschner scalar $K(0) = 4.95\,\text{km}^{-4}$.
