# NVG: академическая феноменологическая версия циклической космологии

> Этот документ представляет академическую версию объединенной модели из `NULL_VECTOR_GRAVITY.md` и `NULL_GRAVITY_CYCLIC_UNIVERSE.md`. Цель текста не в том, чтобы заявить новую подтвержденную фундаментальную теорию, а в том, чтобы перевести исходные идеи в форму физически узнаваемой феноменологической схемы, совместимой с общей теорией относительности, космологией FLRW, скалярными полями и эффективными bounce-моделями.

---

## 1. Аннотация

Модель NVG в строгой версии интерпретируется как эффективная космологическая конструкция, в которой:

- `Ω₀` понимается как вакуумное основное состояние эффективного поля;
- `δ` описывает возмущения плотности или поля относительно фона;
- `𝒲` играет роль скалярного поля или параметра порядка, вносящего вклад в тензор энергии-импульса;
- крупномасштабная динамика задается уравнениями Эйнштейна и Фридмана;
- цикличность вводится через bounce-аппроксимацию, а не через буквальное "сжатие Вселенной в частицу".

В этой записи исходные метафоры сохраняются как интерпретационный слой, но не подменяют собой стандартные уравнения.

## 2. Статус модели

### 2.1. Что утверждается

Модель утверждает только следующее:

1. исходные две концептуальные схемы можно свести к единому эффективному описанию;
2. это описание может быть записано через известные объекты релятивистской космологии;
3. большая часть необычных утверждений должна рассматриваться как интерпретация, а не как установленный физический результат.

### 2.2. Что не утверждается

Модель не утверждает, что:

- философское Ничто является наблюдаемой физической сущностью;
- циклическая космология уже подтверждена наблюдениями;
- темная материя доказанно сводится к скрытым измерениям;
- `G` наблюдаемо меняется от цикла к циклу;
- существует экспериментально подтвержденное мета-время вне пространства-времени.

Такое разграничение обязательно, если текст должен выглядеть как академическая записка, а не как замена существующей космологической модели.

## 4. Минимальный математический каркас

### 4.1. Геометрия фона

Фоновая метрика принимается в стандартной форме FLRW:

```math
ds^2 = -c^2 dt^2 + a^2(t)\,\gamma_{ij} dx^i dx^j
```

где `a(t)` - масштабный фактор, а `\gamma_{ij}` - трехмерная метрика пространства постоянной кривизны.

### 4.2. Динамика фона

Эволюция задается уравнениями Фридмана:

```math
H^2 = \left(\frac{\dot a}{a}\right)^2 = \frac{8\pi G}{3}\rho - \frac{k c^2}{a^2} + \frac{\Lambda c^2}{3}
```

```math
\dot H = -4\pi G\left(\rho + \frac{p}{c^2}\right) + \frac{k c^2}{a^2}
```

```math
\dot\rho + 3H\left(\rho + \frac{p}{c^2}\right) = 0
```

### 4.3. Полное релятивистское уравнение модели

Объединенная модель записывается как ОТО с дополнительными эффективными источниками:

```math
G_{\mu\nu} + \Lambda_{\mathrm{eff}} g_{\mu\nu}
=
\frac{8\pi G_{\mathrm{eff}}}{c^4}
\left(
T^{(m)}_{\mu\nu}
+ T^{(\mathcal{W})}_{\mu\nu}
+ T^{(\mathrm{extra})}_{\mu\nu}
\right)
```

Здесь:

- `T^{(m)}_{\mu\nu}` - тензор обычной материи и излучения;
- `T^{(\mathcal{W})}_{\mu\nu}` - вклад скалярного поля `𝒲`;
- `T^{(\mathrm{extra})}_{\mu\nu}` - эффективный вклад скрытых степеней свободы или дополнительных измерений.

### 4.4. Поле `𝒲`

Поле `𝒲` задается стандартным лагранжианом:

```math
\mathcal{L}_{\mathcal{W}} = -\frac{1}{2}\partial_\mu \mathcal{W}\,\partial^\mu \mathcal{W} - V(\mathcal{W})
```

с потенциалом вида:

```math
V(\mathcal{W}) = V_0 + \frac{m_{\mathcal{W}}^2}{2}(\mathcal{W}-\mathcal{W}_0)^2 + \frac{\lambda}{4}(\mathcal{W}-\mathcal{W}_0)^4
```

В такой записи `Ω₀` интерпретируется как вакуумное значение поля:

```math
\Omega_0 \equiv \mathcal{W}_0
```

Уравнение движения:

```math
\Box \mathcal{W} - \frac{dV}{d\mathcal{W}} = 0
```

Тензор энергии-импульса поля:

```math
T^{(\mathcal{W})}_{\mu\nu}
=
\partial_\mu \mathcal{W}\,\partial_\nu \mathcal{W}
- g_{\mu\nu}
\left[
\frac{1}{2}\partial_\alpha \mathcal{W}\,\partial^\alpha \mathcal{W} + V(\mathcal{W})
\right]
```

### 4.5. Возмущения и структура

Вместо расплывчатого "отклонения от Ничто" используется стандартный контраст плотности:

```math
\delta(x,t) = \frac{\rho(x,t)-\bar\rho(t)}{\bar\rho(t)}
```

Связность паутины лучше всего интерпретировать через двухточечную корреляцию:

```math
\xi(r) = \langle \delta(x)\,\delta(x+r) \rangle
```

```math
\Gamma_{ij} = \frac{\xi(r_{ij})}{\xi(0)}
```

Так образ "паутины" переходит в стандартный язык крупномасштабной структуры.

### 4.6. Слабопольный предел

В квазиньютоновском режиме модель должна сводиться к:

```math
\vec a = -\nabla \Phi
```

```math
\nabla^2 \Phi = 4\pi G_{\mathrm{eff}}\rho_{\mathrm{tot}}
```

где:

```math
\rho_{\mathrm{tot}} = \rho_m + \rho_{\mathcal{W}} + \rho_{\mathrm{extra}}
```

```math
\rho_{\mathcal{W}} = \frac{1}{2}\dot{\mathcal{W}}^2 + \frac{1}{2a^2}|\nabla\mathcal{W}|^2 + V(\mathcal{W})
```

### 4.7. Эффективный вклад скрытых измерений

Если сохранять идею второго текста о многомерно сжатой паутине, то наиболее аккуратная запись выглядит как редукция многомерного поля:

```math
\rho_{\mathrm{extra}}
\sim
\sum_a \frac{1}{2R_a^2}
\left|\frac{\partial \mathcal{W}}{\partial y_a}\right|^2
```

Это не установленная модель темной материи, а допустимая эффективная параметризация скрытого гравитирующего вклада.

## 5. Циклическая динамика

### 5.1. Почему обычного FLRW недостаточно

Обычные уравнения Фридмана сами по себе не гарантируют bounce. Поэтому циклическая версия должна вводиться как эффективное обобщение, а не как прямой вывод из стандартной `\Lambda`CDM-космологии.

### 5.2. Bounce-параметризация

Минимальная феноменологическая запись:

```math
H^2
=
\frac{8\pi G}{3}\rho\left(1-\frac{\rho}{\rho_c}\right)
- \frac{k c^2}{a^2}
+ \frac{\Lambda c^2}{3}
```

где `\rho_c` - критическая плотность bounce-перехода.

Тогда:

- цикл `n` - это промежуток между двумя соседними bounce-событиями;
- `ℓ_n` - минимальный масштаб или максимальная кривизна цикла;
- `Ĉ` - не буквальная компрессия мира в частицу, а эффективный переход через фазу высокой плотности.

### 5.3. Перенос информации между циклами

Формулировка вида `U_n \rightarrow p_{n+1}` в академической версии должна быть ослаблена:

следующий цикл получает не полную "сжатую Вселенную", а эффективные начальные условия, определяемые состоянием предыдущей высокоплотной фазы.

Если требуется информационная верхняя граница, уместнее ссылаться на голографическую оценку:

```math
S \leq \frac{A}{4\ell_P^2}
```

## 6. Переменные `G_eff` и `\Lambda_eff`

### 6.1. Эффективная гравитационная связь

Если модель хочет сохранить идею циклически зависящего `G_eff`, то физически корректнее всего трактовать это через скалярно-тензорный сектор:

```math
G_{\mathrm{eff}}(\phi) = \frac{1}{8\pi F(\phi)}
```

Поздняя Вселенная при этом должна быть близка к почти постоянному `G`, иначе модель быстро вступает в противоречие с локальными тестами гравитации.

### 6.2. Эффективная темная энергия

Если сохранять идею фазовой эволюции темной энергии, то удобнее использовать стандартную CPL-параметризацию:

```math
w(a) = w_0 + w_a(1-a)
```

```math
\rho_{DE}(a)
=
\rho_{DE,0}
a^{-3(1+w_0+w_a)}
\exp\left[-3w_a(1-a)\right]
```

Тогда параметр `τ` можно понимать как вспомогательный параметр, который перенумеровывает космологическую фазу, не заменяя собой наблюдаемое космическое время.

## 7. Черные дыры и порог возврата

Во второй исходной заметке вводился специальный индекс возврата `\chi`. В академической версии его прямой физический аналог - компактность:

```math
\chi(r) = \frac{2GM(r)}{c^2 r}
```

Условие образования горизонта:

```math
\chi(r) \ge 1
```

Поэтому термин "горизонт Ничто" допустим только как интерпретационный синоним горизонта событий в режиме, где локальная компактность достигает релятивистского порога.

## 8. Наблюдательный статус и ограничения

Этот раздел критичен. Именно он определяет, выглядит ли модель серьезной.

### 8.1. Гравитация

Наблюдения LIGO-Virgo-KAGRA по множеству событий слияний черных дыр показывают, что зарегистрированные гравитационные волны хорошо описываются общей теорией относительности, без убедительных признаков отклонений в генерации, распространении или постмерджер-режиме. Следовательно, любая версия NVG обязана воспроизводить стандартную ОТО как минимум в режимах, доступных текущим гравитационно-волновым тестам.

Практическое следствие: NVG нельзя формулировать как сильную модификацию гравитационной динамики на астрофизических масштабах сегодняшней Вселенной. Допустимы только малые эффективные поправки или новые секторы, редуцирующиеся к ОТО в наблюдаемом пределе.

### 8.2. Темная энергия

Новые результаты DESI усилили намеки на возможную эволюцию темной энергии, но это пока не эквивалентно открытию новой физики. Поэтому фазовая функция цикла не должна подаваться как установленный факт. Она может использоваться только как феноменологическая параметризация, которую нужно сверять с `w(a)` и с будущими обзорами крупномасштабной структуры.

Практическое следствие: безопаснее говорить не "`\Lambda` циклически меняется", а "модель допускает эффективную эволюцию сектора темной энергии, совместимую с параметризацией CPL и проверяемую данными DESI и последующих обзоров".

### 8.3. Хабблово напряжение

Данные JWST усилили доверие к локальной лестнице расстояний и тем самым не сняли хабблово напряжение. Однако само по себе это не доказывает ни циклическую космологию, ни модифицированную гравитацию.

Практическое следствие: хабблово напряжение в рамках NVG можно использовать только как мотивацию рассматривать дополнительные поля, раннюю динамическую темную энергию или нетривиальные начальные условия, но не как прямое подтверждение модели.

### 8.4. Переменный `G`

Любая версия модели с переменным `G_eff` обязана проходить строгие ограничения со стороны солнечно-системных тестов, бинарных пульсаров, нуклеосинтеза и реликтового излучения. Поэтому в позднюю эпоху допустимы только очень малые отклонения от практически постоянного значения `G`.

Практическое следствие: в академической записи лучше не давать агрессивных прогнозов на заметный современный дрейф `G`, если отдельно не указана конкретная модель и ее параметрическое окно допустимости.

### 8.5. Темная материя как скрытая геометрия

Гипотеза о том, что темная материя есть эффективная энергия скрытых измерений или сжатых мод `𝒲`, остается спекулятивной. Такая конструкция должна одновременно воспроизводить вращательные кривые галактик, линзирование, рост структуры и космический микроволновый фон. Пока это следует формулировать как исследовательскую возможность, а не как уже согласованную замену стандартной холодной темной материи.

### 8.6. Bounce-космология

Bounce-сценарии обсуждаются в теоретической космологии давно, но в настоящее время не являются наблюдательно установленным описанием ранней Вселенной. Поэтому цикл в NVG нужно вводить как допустимое расширение, которое обязано воспроизвести хотя бы те наблюдательные свойства, которые успешно описывает инфляционная `\Lambda`CDM-модель.

## 9. Фальсифицируемые формы модели

Если из NVG делать исследовательскую программу, а не только концептуальный текст, то проверяемые утверждения должны формулироваться в слабой и измеримой форме.

Подходящие формулировки:

1. поздняя космология допускает малое, но измеримое отклонение `w(a)` от `-1`;
2. поле `𝒲` или родственный скалярный сектор оставляет вклад в рост структуры, не нарушая гравитационно-волновые тесты ОТО;
3. bounce-сценарий может оставлять специфические низкочастотные или крупномасштабные сигнатуры, но только в рамках уже допустимых ограничений CMB и LSS;
4. extra-dimensional вклад допустим только тогда, когда он проявляется гравитационно, но не конфликтует с линзированием и эволюцией возмущений.

Неподходящие формулировки:

- "гравитация доказанно есть тяга к Ничто";
- "темная материя на самом деле уже объяснена скрытыми измерениями";
- "DESI и JWST подтвердили цикл Вселенной".

## 10. Консолидированная итоговая запись

Если оставить только математически и методологически устойчивое ядро, то итоговая академическая формула модели такова:

```math
\boxed{
G_{\mu\nu} + \Lambda_{\mathrm{eff}} g_{\mu\nu}
=
\frac{8\pi G_{\mathrm{eff}}}{c^4}
\left(
T^{(m)}_{\mu\nu}
```

где интерпретация символов задается так:

- `Ω₀` - вакуумный минимум поля `𝒲`;

- `Γᵢⱼ` - корреляционная связность структуры;
- `Ĉ` - эффективный bounce-переход;
- темная материя и скрытые измерения при желании входят через `T^{(\mathrm{extra})}_{\mu\nu}`, но не объявляются установленным фактом.

## 11. Итоговая академическая формулировка

В академическом стиле модель NVG разумно формулировать так:

> NVG представляет собой не альтернативу общей теории относительности как таковой, а эффективную феноменологическую надстройку, в которой исходные метафоры "Ничто", "паутины" и "циклов" переводятся в язык вакуумного состояния, скалярного поля, корреляций крупномасштабной структуры и bounce-динамики. При этом основная гравитационная динамика по-прежнему задается уравнениями Эйнштейна, а все нестандартные элементы вводятся только как дополнительные эффективные секторы, которые обязаны редуцироваться к стандартной космологии и стандартной гравитации в уже проверенных наблюдательных пределах.

Эта формулировка позволяет сохранить содержательное ядро двух исходных документов, но убирает места, где художественный образ начинал выдавать себя за уже установленную физику.

- как феноменологическая модель, а не как объявленная замена современной физики;
- как расширение ОТО дополнительными эффективными полями, а не как отказ от нее;
- как циклический сценарий с bounce-параметризацией, а не как буквально доказанное многомировое сжатие;
- как интерпретационная рамка, которая обязана проходить через реальные наблюдательные ограничения.

В такой форме две исходные теории действительно можно свести в один серьезный документ без внутреннего конфликта между метафорой и физикой.
