# NVG: согласованная феноменологическая модель циклической космологии

> Этот документ объединяет идеи из `NULL_VECTOR_GRAVITY.md` и `NULL_GRAVITY_CYCLIC_UNIVERSE.md`, но переводит их из художественно-метафорического языка в максимально физически согласованную форму. Ниже не вводится новая подтвержденная теория природы. Вместо этого дается единая феноменологическая схема, опирающаяся на существующие уравнения общей теории относительности, космологии, скалярных полей и эффективных bounce-моделей.

---

## 1. Принцип объединения

Обе исходные заметки говорят об одном и том же на двух уровнях:

- есть базовое состояние `Ω₀`, к которому система стремится вернуться;
- есть первичное смещение `τ`, запускающее эволюцию;
- материя трактуется как локальное отклонение `δ` от базового состояния;
- гравитация возникает не как отдельная магическая сила, а как следствие геометрии и распределения этих отклонений;
- Вселенная допускает цикличность, где один цикл задает начальные условия следующего.

Чтобы это не расходилось с реальной физикой, нужно сделать главное переопределение:

`Ω₀` следует понимать не как буквальное философское Ничто, а как вакуумное основное состояние эффективного поля или геометрии.

Тогда язык обеих записок можно свести к стандартной схеме:

1. есть фоновая метрика и масштабный фактор;
2. есть поле порядка или скалярное поле `𝒲`, описывающее состояние вакуума;
3. есть возмущения плотности `δ`, из которых формируются структура и гравитация;
4. есть эффективная динамика цикла, задаваемая уравнениями Фридмана или их bounce-обобщением.

## 2. Словарь соответствий между двумя документами и физикой

| Символ | В исходных текстах | Физически согласованная интерпретация |
|--------|--------------------|----------------------------------------|
| `Ω₀` | Абсолютное Ничто | Вакуумное основное состояние, минимум эффективного потенциала |
| `τ` | Мета-время, первичный вектор | Параметр цикла или конформное/эффективное космологическое время |
| `δ(x,τ)` | Отклонение от Ничто | Контраст плотности или возмущение поля |
| `𝒲` | Паутина связей | Эффективное скалярное поле, параметр порядка или поле крупномасштабной структуры |
| `Γᵢⱼ` | Связность паутины | Корреляционная функция структуры `ξ(r)` или ядро связности среды |
| `G_eff` | Гравитационная постоянная цикла | Эффективная гравитационная связь в скалярно-тензорной модели |
| `ℓₙ` | Планковское зерно цикла | Минимальный масштаб или радиус кривизны в bounce-фазе цикла `n` |
| `Ĉ` | Компрессия цикла | Эффективный bounce/сжатие с переносом начальных условий |

Ключевой шаг здесь такой: обе исходные формулировки становятся совместимыми, если считать, что они описывают не буквальную новую механику, а один и тот же эффективный слой поверх уже известных уравнений.

## 3. Базовая геометрия: стандартный космологический каркас

Если теория должна быть связана с реальной физикой, ее фон должен задаваться метрикой Фридмана-Леметра-Робертсона-Уокера:

```math
ds^2 = -c^2 dt^2 + a^2(t)\,\gamma_{ij} dx^i dx^j
```

где:

- `t` - космическое время;
- `a(t)` - масштабный фактор;
- `\gamma_{ij}` - пространственная метрика постоянной кривизны.

Тогда динамика фона задается уравнениями Фридмана:

```math
H^2 = \left(\frac{\dot a}{a}\right)^2 = \frac{8\pi G}{3}\rho - \frac{k c^2}{a^2} + \frac{\Lambda c^2}{3}
```

```math
\dot H = -4\pi G\left(\rho + \frac{p}{c^2}\right) + \frac{k c^2}{a^2}
```

и законом сохранения:

```math
\dot\rho + 3H\left(\rho + \frac{p}{c^2}\right) = 0
```

Это каркас, в который нужно встраивать все образы из двух документов. Без него теория остается литературной. С ним она становится феноменологической.

## 4. Как правильно переопределить `Ω₀`

Вместо метафизического Ничто вводится эффективное поле `𝒲`, у которого есть вакуумное состояние `𝒲 = 𝒲₀`. Именно это состояние и удобно обозначить как `Ω₀`:

```math
\Omega_0 \equiv \mathcal{W}_0
```

Тогда фраза "все стремится вернуться к Ничто" получает физический смысл:

система стремится к минимуму потенциала.

Для этого достаточно стандартного лагранжиана скалярного поля:

```math
\mathcal{L}_{\mathcal{W}} = -\frac{1}{2}\partial_\mu \mathcal{W}\,\partial^\mu \mathcal{W} - V(\mathcal{W})
```

с потенциалом вида:

```math
V(\mathcal{W}) = V_0 + \frac{m_{\mathcal{W}}^2}{2}(\mathcal{W}-\mathcal{W}_0)^2 + \frac{\lambda}{4}(\mathcal{W}-\mathcal{W}_0)^4
```

Это напрямую согласует идею из `NULL_VECTOR_GRAVITY.md`, где возврат к `Ω₀` описывался через квадратичный и квартичный члены.

Уравнение движения тогда стандартно:

```math
\Box \mathcal{W} - \frac{dV}{d\mathcal{W}} = 0
```

Именно так художественная формула "возвращения к Ничто" превращается в обычную релаксацию поля к минимуму потенциала.

## 5. Что такое `δ`: корректная замена для отклонения от Ничто

В реальной космологии уже есть естественная величина для "отклонения" - контраст плотности:

```math
\delta(x,t) = \frac{\rho(x,t)-\bar\rho(t)}{\bar\rho(t)}
```

где `\bar\rho(t)` - средняя плотность Вселенной.

Тогда обе исходные идеи совпадают:

- из `NULL_GRAVITY_CYCLIC_UNIVERSE.md`: частица есть локальное напряжение или сгусток;
- из `NULL_VECTOR_GRAVITY.md`: объект есть отклонение `δ` от базового состояния.

В физическом языке это значит одно и то же:

массивные структуры - это области с `\delta > 0`, то есть сверхплотные возмущения на фоне `\bar\rho`.

## 6. Гравитация без новой мистической силы

Чтобы две заметки не конфликтовали с общей теорией относительности, гравитацию лучше не записывать как совершенно новый фундаментальный закон. Ее надо понимать так же, как в современной физике:

в полной теории гравитация - это кривизна пространства-времени, а в слабом поле - потенциал `\Phi`.

Слабопольный предел:

```math
\vec a = -\nabla \Phi
```

```math
\nabla^2 \Phi = 4\pi G\rho_{\text{tot}}
```

Если хотим сохранить "паутину" и скрытые вклады из второго документа, то просто расширяем полную плотность:

```math
\rho_{\text{tot}} = \rho_m + \rho_{\mathcal{W}} + \rho_{\text{extra}}
```

где:

```math
\rho_{\mathcal{W}} = \frac{1}{2}\dot{\mathcal{W}}^2 + \frac{1}{2a^2}|\nabla \mathcal{W}|^2 + V(\mathcal{W})
```

Это объединяет две исходные записи сразу:

- из `NULL_GRAVITY_CYCLIC_UNIVERSE.md`: `\rho \sim |\nabla \mathcal{W}|^2`;
- из `NULL_VECTOR_GRAVITY.md`: добавочный вклад `W_{\mu\nu}(\mathcal{W})` в гравитацию.

В результате не нужна новая отдельная "сила тяги к Ничто". Достаточно сказать:

гравитация создается суммарным тензором энергии-импульса материи, поля `𝒲` и возможных скрытых степеней свободы.

## 7. Полная релятивистская запись

Самая чистая объединяющая формула для обеих заметок выглядит так:

```math
G_{\mu\nu} + \Lambda_{\mathrm{eff}} g_{\mu\nu}
=
\frac{8\pi G_{\mathrm{eff}}}{c^4}
\left(
T^{(m)}_{\mu\nu}
+ T^{(\mathcal{W})}_{\mu\nu}
+ T^{(\mathrm{extra})}_{\mu\nu}
\right)
```

где тензор поля `𝒲` стандартен:

```math
T^{(\mathcal{W})}_{\mu\nu}
=
\partial_\mu \mathcal{W}\,\partial_\nu \mathcal{W}
-
g_{\mu\nu}
\left[
\frac{1}{2}\partial_\alpha \mathcal{W}\,\partial^\alpha \mathcal{W}
+ V(\mathcal{W})
\right]
```

Эта запись делает две вещи:

1. сохраняет идею из первого документа, что кроме обычной материи есть еще структурный вклад паутины;
2. сохраняет идею из второго документа, что крупномасштабная геометрия зависит не только от видимой материи.

Но теперь это не выдуманная формула, а стандартная форма ОТО с дополнительными источниками.

## 8. Что делать с `G_eff` и переменной `Λ`

В первом документе гравитационная постоянная и космологический член зависят от цикла и фазы. В реальной физике ближайший к этому язык - скалярно-тензорные теории и динамическая темная энергия.

### 8.1. Эффективная гравитационная связь

Вместо ручной формулы для `G_eff(n)` лучше использовать стандартную запись:

```math
G_{\mathrm{eff}}(\phi) = \frac{1}{8\pi F(\phi)}
```

где `F(\phi)` - функция дополнительного скалярного поля в духе Бранса-Дикке или более общих scalar-tensor моделей.

Тогда зависимость от цикла можно трактовать так:

разные циклы стартуют с разными эффективными начальными значениями поля `\phi`, а значит и с немного разным `G_{\mathrm{eff}}`.

Это согласует идею `G_eff(n)` с реальной математикой лучше, чем прямая замена ньютоновской постоянной по номеру цикла.

### 8.2. Переменная темная энергия

Вместо искусственной функции `\Lambda(\tau) = \Lambda_0 \cos(2\pi \tau/T)` физически корректнее использовать стандартную параметризацию CPL:

```math
w(a) = w_0 + w_a(1-a)
```

```math
\rho_{DE}(a)
=
\rho_{DE,0}
a^{-3(1+w_0+w_a)}
\exp\left[-3w_a(1-a)\right]
```

Если нужен именно язык фаз цикла, можно считать, что параметр `\tau` просто параметризует эволюцию `w(a)` или `\rho_{DE}(a)`. Тогда художественный образ сохраняется, а математика остается стандартной.

## 9. Как объединить `𝒲` и `Γᵢⱼ`: паутина как корреляционная структура

В первом документе `Γᵢⱼ(𝒲)` отвечает за связность паутины. Во втором `𝒲` - это поле, на котором сидят узлы материи. Эти две идеи можно совместить без конфликта, если связность определить через корреляции поля или плотности.

Стандартная двухточечная функция корреляции:

```math
\xi(r) = \langle \delta(x)\,\delta(x+r) \rangle
```

Тогда нормированную связность можно записать как:

```math
\Gamma_{ij} = \frac{\xi(r_{ij})}{\xi(0)}
```

или как корреляцию самого поля:

```math
\Gamma_{ij} = \frac{\langle \mathcal{W}(x_i)\mathcal{W}(x_j) \rangle}{\langle \mathcal{W}^2 \rangle}
```

Это важный шаг: "паутина" перестает быть чисто образной и становится эквивалентом наблюдаемой крупномасштабной структуры или коррелированного фонового поля.

## 10. Темная материя и скрытые измерения: как сделать это совместимым с физикой

Во втором документе темная материя трактуется как сжатая паутина в скрытых измерениях. В существующей теоретической физике наиболее близкий язык - компактные дополнительные измерения и эффективные моды Калуцы-Клейна.

Тогда вклад скрытых измерений можно записать как эффективную добавку к плотности:

```math
\rho_{\mathrm{extra}}
\sim
\sum_a \frac{1}{2R_a^2}
\left|\frac{\partial \mathcal{W}}{\partial y_a}\right|^2
```

где `R_a` - характерный радиус компактификации скрытого измерения `y_a`.

Это не доказанная модель темной материи. Но это уже физически узнаваемая формулировка:

- не вводится чистая метафизика;
- сохраняется идея скрытого гравитирующего вклада;
- формула выглядит как обычная редукция многомерного поля к эффективной 4D-теории.

То есть лучшая строгая версия утверждения такая:

темная материя в этой модели - не новый сорт вещества по определению, а эффективная гравитирующая энергия не наблюдаемых напрямую степеней свободы.

## 11. Циклы Вселенной: как заменить чистую поэтику реальной динамикой

Чтобы идея циклов была физически связной, ее нужно привязать не к литературному "сжатию в точку", а к известным bounce-сценариям. Самая удобная эффективная форма:

```math
H^2
=
\frac{8\pi G}{3}\rho\left(1-\frac{\rho}{\rho_c}\right)
- \frac{k c^2}{a^2}
+ \frac{\Lambda c^2}{3}
```

Здесь `\rho_c` - критическая плотность, при достижении которой стандартное сжатие переходит в bounce.

Это дает прямую интерпретацию циклам:

- цикл `n` - промежуток между двумя соседними bounce-событиями;
- `ℓ_n` - минимальный характерный масштаб цикла, связанный с `a_{\min}^{(n)}` или максимальной кривизной;
- операция `Ĉ` - не магическая компрессия, а эффективный переход через фазу максимальной плотности.

Тогда формула из второго документа

```math
U_n \rightarrow p_{n+1}
```

получает более строгий смысл:

не "вся Вселенная стала одной частицей", а информация о состоянии цикла `n` задает эффективные начальные условия цикла `n+1`.

## 12. Как корректно говорить об информации предыдущих циклов

Первый документ использует сильную гипотезу о том, что каждый планковский объем содержит предыдущую Вселенную. В физически осторожной версии это лучше заменить на голографическую границу информации:

```math
S \leq \frac{A}{4\ell_P^2}
```

где `S` - энтропия, `A` - площадь ограничивающей поверхности.

Тогда аккуратная формулировка звучит так:

информация предыдущего цикла не обязана буквально храниться в каждой точке, но может быть ограничена голографическим верхним пределом и передаваться через эффективные начальные условия после bounce-фазы.

Так сохраняется смысл исходной идеи, но убирается утверждение, которое пока невозможно защитить физически.

## 13. Черные дыры: как перевести индекс возврата в строгий язык

Во втором документе вводится индекс `\chi`, при превышении порога которого возникает горизонт Ничто. В реальной общей теории относительности прямой аналог уже существует: компактность объекта.

```math
\chi(r) = \frac{2GM(r)}{c^2 r}
```

Горизонт возникает при условии:

```math
\chi(r) \ge 1
```

Это почти идеальная замена исходной идеи:

- не нужно вводить новый мистический критерий;
- сохраняется смысл "точки, где геометрия перестает выпускать свет";
- язык остается совместимым с черными дырами Шварцшильда и более общими решениями ОТО.

Если хочется сохранить исходную терминологию, можно писать так:

"горизонт Ничто" в этой модели - это художественное имя для области, где локальная компактность достигает релятивистского порога образования горизонта.

## 14. Единая формула, которая реально объединяет оба текста

Если оставить только то, что можно защитить математически, объединенное ядро выглядит так:

```math
G_{\mu\nu} + \Lambda_{\mathrm{eff}} g_{\mu\nu}
=
\frac{8\pi G_{\mathrm{eff}}}{c^4}
\left(
T^{(m)}_{\mu\nu}
+ T^{(\mathcal{W})}_{\mu\nu}
+ T^{(\mathrm{extra})}_{\mu\nu}
\right)
```

с ньютоновским пределом

```math
\nabla^2 \Phi = 4\pi G_{\mathrm{eff}}\left(\rho_m + \rho_{\mathcal{W}} + \rho_{\mathrm{extra}}\right)
```

и определениями

```math
\delta = \frac{\rho-\bar\rho}{\bar\rho}
```

```math
\Gamma_{ij} = \frac{\xi(r_{ij})}{\xi(0)}
```

```math
\rho_{\mathcal{W}} = \frac{1}{2}\dot{\mathcal{W}}^2 + \frac{1}{2a^2}|\nabla\mathcal{W}|^2 + V(\mathcal{W})
```

```math
\rho_{\mathrm{extra}} \sim \sum_a \frac{1}{2R_a^2}\left|\frac{\partial \mathcal{W}}{\partial y_a}\right|^2
```

А циклическая динамика задается отдельным уравнением фона:

```math
H^2
=
\frac{8\pi G}{3}\rho\left(1-\frac{\rho}{\rho_c}\right)
- \frac{k c^2}{a^2}
+ \frac{\Lambda c^2}{3}
```

Это и есть точка, где обе исходные теории сходятся без внутреннего конфликта.

## 15. Что в этой версии является физикой, а что интерпретацией

### Можно считать физически корректным языком

- метрику FLRW;
- уравнения Фридмана;
- уравнения Эйнштейна с дополнительными полями;
- скалярное поле с потенциалом;
- слабопольный предел `\vec a = -\nabla \Phi`;
- компактность `\chi = 2GM/(c^2 r)` для горизонтов;
- эффективные extra-dimensional вклады в стиле Калуцы-Клейна;
- bounce-уравнение как феноменологическую форму цикличности.

### Остается интерпретацией, а не установленным фактом

- отождествление вакуума с философским Ничто;
- перенос информации между циклами в сильной форме;
- трактовка темной материи именно как скрытых складок `𝒲`;
- зависимость `G` от номера цикла как наблюдаемый факт;
- буквальное существование мета-времени вне пространства-времени.

Именно такое разделение делает итоговый документ серьезнее: он не маскирует метафору под готовую физику, но и не теряет исходную идею.

## 16. Финальная формулировка для дальнейшей работы

Если нужна одна компактная версия теории, пригодная и для текста, и для дальнейшей доработки, лучше использовать такую формулировку:

> `Ω₀` - это не буквальное Ничто, а вакуумный минимум эффективного поля `𝒲`. Материя и структура Вселенной задаются возмущениями `δ` относительно этого минимума. Гравитация описывается обычной кривизной пространства-времени через уравнения Эйнштейна, но в источник кривизны, кроме видимой материи, входят энергия поля `𝒲` и возможные скрытые степени свободы. Циклическая космология задается не поэтическим "схлопыванием в точку", а bounce-динамикой фона, где один цикл передает следующему не мистический остаток, а эффективные начальные условия.

В совсем короткой математической форме:

```math
\boxed{
G_{\mu\nu} + \Lambda_{\mathrm{eff}} g_{\mu\nu}
=
\frac{8\pi G_{\mathrm{eff}}}{c^4}
\left(
T^{(m)}_{\mu\nu}
+ T^{(\mathcal{W})}_{\mu\nu}
+ T^{(\mathrm{extra})}_{\mu\nu}
\right),
\qquad
H^2
=
\frac{8\pi G}{3}\rho\left(1-\frac{\rho}{\rho_c}\right)
- \frac{k c^2}{a^2}
+ \frac{\Lambda c^2}{3}
}
```

Это уже не фантастическая формула "гравитации Ничто", а физически узнаваемая феноменологическая модель, в которой исходные образы двух документов сведены в одну согласованную систему.
