# Гравитация Нулевого Вектора: единая феноменологическая модель

> Объединённая статья на основе семи рабочих документов проекта NVG. Текст сводит философскую мотивацию, полевую теорию, циклическую космологию, структуру массы адронов и наблюдательные ограничения в одну согласованную конструкцию.

---

## Аннотация

Предлагается феноменологическая модель, в которой физически наблюдаемая материя, гравитация и масса интерпретируются как проявления локальных отклонений от вакуумного основного состояния `Ω₀`. Модель строится как расширение общей теории относительности каноническим скалярным полем `𝒲` и допускает циклическую космологию через bounce-динамику. На микроскопическом уровне язык вакуумного отклонения связывается с непертурбативной КХД через сигма-термы и аномальные вклады в массу адронов.

Центральные результаты:

1. В слабопольном пределе теория воспроизводит обобщённое уравнение Пуассона с полной вакуумно-субтрагированной плотностью.
2. Локальная скорость света остаётся инвариантной и равной `c`.
3. Энергия покоя выражается формулой `E_Ω = M_Ω c²`, где `M_Ω` — масса вакуумного отклонения.
4. Для нуклона доля массы, не сводимая к токовым кварковым массам, составляет `f_Ω ≈ 0.91`.

Стандартные формулы Ньютона, Эйнштейна и `E = mc²` возникают как строгие пределы.

## 1. Философская основа и мотивация

### 1.1. Аксиома Нуля

Единственное истинно стабильное состояние — **Ничто** (`Ω₀`). Всё, что существует, есть отклонение от этого состояния. Гравитация — стремление вернуться.

Вселенная не «появилась из ничего» — она **является** возмущением Ничего. Каждая частица, каждое поле — это натяжение на ткани `Ω₀`, стремящееся к разрядке.

### 1.2. Первичный вектор

Мета-время `τ` смещает `Ω₀`, создавая первое измерение — направление нестабильности:

```math
\hat{T}|\Omega_0\rangle = \varepsilon\,|1\rangle, \qquad \varepsilon \to 0^+.
```

### 1.3. Паутина и частицы

Отклонения не изолированы — они связаны топологической паутиной `𝒲`. Каждая частица — локальное напряжение этой паутины:

```math
p_i = \delta\Omega_i
```

Частица стремится не в пустоту, а к ближайшим участкам, где Ничто сильнее сжато. Так возникает гравитация.

### 1.4. Переход к физическому языку

Для научной состоятельности метафоры требуют переопределения:

| Символ | Художественный смысл | Физическая интерпретация |
|--------|---------------------|--------------------------|
| `Ω₀` | Абсолютное Ничто | Вакуумный минимум потенциала поля `𝒲` |
| `τ` | Мета-время | Параметр цикла или эффективное космологическое время |
| `δ(x,t)` | Отклонение от Ничто | Контраст плотности `(ρ - ρ̄)/ρ̄` |
| `𝒲` | Паутина связей | Эффективное скалярное поле |
| `Γᵢⱼ` | Связность паутины | Корреляционная функция `ξ(r)/ξ(0)` |
| `G_eff` | Постоянная цикла | Скалярно-тензорная связь |
| `Ĉ` | Компрессия вселенной | Эффективный bounce-переход |

## 2. Действие и полевые уравнения

### 2.1. Полное действие

```math
S
=
\int d^4x\,\sqrt{-g}
\left[
\frac{c^4}{16\pi G_*}(R - 2\Lambda_{\mathrm{eff}})
- \frac{1}{2}\partial_\mu \mathcal{W}\,\partial^\mu \mathcal{W}
- V(\mathcal{W})
+ \mathcal{L}_m
+ \mathcal{L}_{\mathrm{extra}}
\right].
```

Здесь `G_*` — наблюдаемая гравитационная связь, а вакуумное состояние определяется как:

```math
\Omega_0 \equiv \mathcal{W}_0,
\qquad
\left.\frac{dV}{d\mathcal{W}}\right|_{\mathcal{W}_0} = 0,
\qquad
\left.\frac{d^2V}{d\mathcal{W}^2}\right|_{\mathcal{W}_0} > 0.
```

Потенциал четвёртого порядка:

```math
V(\mathcal{W}) = V_0 + \frac{m_{\mathcal{W}}^2}{2}(\mathcal{W}-\mathcal{W}_0)^2 + \frac{\lambda}{4}(\mathcal{W}-\mathcal{W}_0)^4.
```

### 2.2. Уравнения Эйнштейна с расширенным источником

Вариация по метрике:

```math
\boxed{
G_{\mu\nu} + \Lambda_{\mathrm{eff}} g_{\mu\nu}
=
\frac{8\pi G_*}{c^4}
\left(
T^{(m)}_{\mu\nu}
+ T^{(\mathcal{W})}_{\mu\nu}
+ T^{(\mathrm{extra})}_{\mu\nu}
\right)
}
```

где тензор энергии-импульса поля:

```math
T^{(\mathcal{W})}_{\mu\nu}
=
\partial_\mu \mathcal{W}\,\partial_\nu \mathcal{W}
- g_{\mu\nu}
\left[
\frac{1}{2}\partial_\alpha \mathcal{W}\,\partial^\alpha \mathcal{W}
+ V(\mathcal{W})
\right].
```

Вариация по `𝒲`:

```math
\Box \mathcal{W} - \frac{dV}{d\mathcal{W}} = 0.
```

Это стандартная ОТО с дополнительным скалярным сектором. Тензор Эйнштейна не изменён, причинный конус не изменён, геодезическая структура стандартна.

### 2.3. Вакуумно-субтрагированная плотность

Физические состояния — отклонения от вакуума. Вводим:

```math
\varepsilon_{\Omega}(x)
=
\rho_m c^2
+ \rho_{\mathrm{extra}} c^2
+ \frac{1}{2}\dot{\mathcal{W}}^2
+ \frac{1}{2a^2}|\nabla \mathcal{W}|^2
+ V(\mathcal{W}) - V(\mathcal{W}_0).
```

Масса отклонения:

```math
M_{\Omega} = \frac{1}{c^2}\int d^3x\,\varepsilon_{\Omega}(x).
```

## 3. Три основных результата

### 3.1. Гравитация: ньютоновский предел

В статическом слабом поле (`|Φ/c²| ≪ 1`) теория сводится к:

```math
\nabla^2 \Phi = 4\pi G_* \rho_{\Omega},
\qquad
\vec{a} = -\nabla \Phi.
```

Для сферически-симметричного источника:

```math
g(r) = -\frac{G_* M_{\Omega}(r)}{r^2},
\qquad
M_{\Omega}(r) = 4\pi\int_0^r \rho_{\Omega}(r')\,r'^2\,dr'.
```

При `𝒲 → 𝒲₀` и `ρ_extra → 0` точно восстанавливается закон Ньютона.

### 3.2. Свет: инвариантность скорости

Электромагнитное поле минимально связано с той же метрикой:

```math
S_{\mathrm{em}} = -\frac{1}{4}\int d^4x\,\sqrt{-g}\,F_{\mu\nu}F^{\mu\nu}.
```

В геометрооптическом пределе фотоны распространяются по нулевым геодезическим (`k_μ k^μ = 0`). В любой локальной инерциальной системе:

```math
\left|\frac{d\mathbf{X}}{dT}\right| = c.
```

Координатная скорость в слабом поле:

```math
v_{\mathrm{coord}} \simeq c\left(1 + \frac{2\Phi}{c^2}\right),
```

что даёт стандартную задержку Шапиро и угол отклонения:

```math
\hat{\alpha} \simeq \frac{4G_* M_{\Omega}}{b\,c^2}.
```

Та же масса `M_Ω`, что управляет свободным падением, управляет и линзированием.

### 3.3. Энергия: обобщённая формула

Для стационарных систем:

```math
\boxed{E_{\Omega} = M_{\Omega}\,c^2}
```

При пренебрежимо малом вкладе `𝒲` и скрытого сектора:

```math
M_{\Omega} \to m, \qquad E_{\Omega} \to mc^2.
```

Для движущегося объекта:

```math
E_{\Omega}(v) = \gamma M_{\Omega}\,c^2, \qquad \gamma = \frac{1}{\sqrt{1-v^2/c^2}}.
```

Формула `E = mc²` — частный случай. Новизна не в коэффициенте `c²`, а в определении массы как интегральной меры отклонения от вакуума.

## 4. Космологический каркас

### 4.1. Фоновая метрика

```math
ds^2 = -c^2 dt^2 + a^2(t)\,\gamma_{ij}\,dx^i dx^j
```

### 4.2. Уравнения Фридмана

```math
H^2 = \frac{8\pi G}{3}\rho - \frac{kc^2}{a^2} + \frac{\Lambda c^2}{3}
```

```math
\dot\rho + 3H\left(\rho + \frac{p}{c^2}\right) = 0
```

Полная плотность:

```math
\rho_{\mathrm{tot}} = \rho_m + \rho_{\mathcal{W}} + \rho_{\mathrm{extra}}
```

### 4.3. Темная энергия

Вместо искусственного `Λ(τ) = Λ₀ cos(2πτ/T)` используется стандартная CPL-параметризация:

```math
w(a) = w_0 + w_a(1-a)
```

```math
\rho_{DE}(a)
=
\rho_{DE,0}\,
a^{-3(1+w_0+w_a)}
\exp\left[-3w_a(1-a)\right]
```

Данные DESI (2025) усилили намёки, что `w ≠ -1`. В рамках NVG это естественно: параметр `τ` перенумеровывает космологическую фазу, допуская эволюцию тёмной энергии.

## 5. Циклическая динамика

### 5.1. Bounce-параметризация

```math
\boxed{
H^2
=
\frac{8\pi G}{3}\rho\left(1-\frac{\rho}{\rho_c}\right)
- \frac{kc^2}{a^2}
+ \frac{\Lambda c^2}{3}
}
```

где `ρ_c` — критическая плотность bounce-перехода.

Цикл `n` — промежуток между двумя bounce-событиями. Формула `U_n → p_{n+1}` получает строгий смысл: не «вся Вселенная стала частицей», а информация цикла `n` задаёт начальные условия цикла `n+1`.

### 5.2. Перенос информации

Голографическая верхняя граница:

```math
S \leq \frac{A}{4\ell_P^2}
```

Информация предыдущего цикла передаётся через эффективные начальные условия, а не хранится буквально в каждой точке.

### 5.3. Схема цикла

```math
\Omega_0
\xrightarrow{\vec{\tau}}
U_n
\xrightarrow{\text{расширение}}
\text{пик}
\xrightarrow{\text{сжатие}}
\hat{C}
\xrightarrow{\text{bounce}}
U_{n+1}
```

## 6. Тёмная материя и скрытые измерения

Если сохранять идею многомерно сжатой паутины, физически корректная запись использует редукцию в стиле Калуцы-Клейна:

```math
\rho_{\mathrm{extra}}
\sim
\sum_a \frac{1}{2R_a^2}
\left|\frac{\partial \mathcal{W}}{\partial y_a}\right|^2
```

где `R_a` — характерный радиус компактификации скрытого измерения `y_a`.

Это не доказанная модель тёмной материи, а допустимая эффективная параметризация: тёмная материя как эффективная гравитирующая энергия ненаблюдаемых напрямую степеней свободы.

## 7. Чёрные дыры

Индекс возврата `χ` из художественной модели имеет прямой аналог — компактность:

```math
\chi(r) = \frac{2GM(r)}{c^2 r}
```

Горизонт возникает при `χ(r) ≥ 1`. «Горизонт Ничто» — интерпретационный синоним горизонта событий, где локальная компактность достигает релятивистского порога.

## 8. Масса адронов: связь с КХД

### 8.1. Проблема массы протона

Масса протона ~938 МэВ. Токовые массы кварков (u, d) — менее 10 МэВ в сумме. Откуда остальные ~90%?

Ответ стандартной КХД: из конфайнмента, глюонной энергии, хирального нарушения симметрии и трековой аномалии. Тензор энергии-импульса КХД:

```math
T^{\mu}{}_{\mu}
=
\frac{\beta(g)}{2g}
G^a_{\alpha\beta}G^{a\alpha\beta}
+
\bigl(1+\gamma_m(g)\bigr)
\sum_q m_q\,\bar q q.
```

Даже при нулевых кварковых массах глюонный член остаётся — масса генерируется непертурбативно.

### 8.2. Сигма-термы и остаточная масса

Через теорему Фейнмана-Хеллмана определяем:

```math
\sigma_{qH} = m_q\,\frac{\partial M_H}{\partial m_q}
```

Остаточная масса вакуумного отклонения адрона:

```math
M_\Omega^{(H)}[\mathcal{F}] \equiv M_H - \sum_{q\in\mathcal{F}} \sigma_{qH}
```

Безразмерная доля:

```math
\boxed{
f_\Omega^{(H)}[\mathcal{F}] = 1 - \frac{1}{M_H}\sum_{q\in\mathcal{F}} \sigma_{qH}
}
```

### 8.3. Оценка для нуклона

Используя решёточные данные:

```math
\sigma_{\pi N} \approx 44\ \mathrm{МэВ}, \qquad
\sigma_{sN} \approx 30\ \mathrm{МэВ}, \qquad
\sum_q \sigma_{qN} \approx 80 \pm 15\ \mathrm{МэВ}
```

Получаем:

```math
f_\Omega^{(N)} \approx 1 - \frac{80 \pm 15}{939} \approx 0.91 \pm 0.02
```

**~91% массы нуклона — это не масса кварков, а структурированная непертурбативная энергия КХД.**

### 8.4. Связь с декомпозицией Цзи

Масса адрона распределяется по секторам:

```math
M_H = M_H^{\mathrm{q,kin}} + M_H^{\mathrm{g}} + M_H^{\mathrm{q,mass}} + M_H^{\mathrm{anom}}
```

В языке NVG:

```math
M_\Omega^{(H)} \sim M_H^{\mathrm{g}} + M_H^{\mathrm{anom}} + M_H^{\mathrm{conf}}
```

Это феноменологическое отождествление, а не теорема. Его ценность — в организации: одна переменная `f_Ω` сжимает вопрос «какая доля массы не сводится к токовым массам» в единое число, сравнимое между адронами.

### 8.5. Наблюдательные каналы

| Канал | Что измеряет |
|-------|-------------|
| Решёточная КХД | Сигма-термы, глюонные доли, аномальные вклады |
| Фотопроизводство J/ψ вблизи порога | Скалярная глюонная структура нуклона |
| Нейтронные звёзды | Параметр `κ_Ω(n_B) = d ln M_Ω / d ln n_B` — отклик вакуумной массы на плотность |

## 9. Наблюдательные ограничения и статус модели

### 9.1. Гравитация

LIGO-Virgo-KAGRA (O4a, 42 слияния ЧД) не обнаружили отклонений от ОТО. Любая версия NVG обязана воспроизводить стандартную ОТО в уже проверенных режимах.

Скорость тензорных мод: `|c_T/c − 1| ≲ 10⁻¹⁵` (GW170817).

### 9.2. Солнечная система

Ограничение Кассини: `|γ_PPN − 1| < 2.3 × 10⁻⁵`. Дрейф `G`: `|Ġ/G| ≲ 10⁻¹³ год⁻¹`.

### 9.3. Ранняя Вселенная

Planck 2018: `H₀ = 67.4 ± 0.5`, `Ωₘ = 0.315 ± 0.007`, `N_eff = 2.99 ± 0.17`. Любые нестандартные вклады до рекомбинации должны быть пертурбативно малы.

### 9.4. Поздняя Вселенная

DESI 2025 усилил намёки на `w ≠ −1`. JWST подтвердил точность локальной лестницы расстояний. Хабблово напряжение не снято — мотивация для дополнительных полей, но не доказательство NVG.

## 10. Что модель утверждает и чего не утверждает

### Утверждается

1. Исходные метафоры сводимы к единому эффективному описанию через стандартные объекты релятивистской космологии.
2. В слабом поле воспроизводятся закон Ньютона, инвариантность скорости света, формула `E = mc²`.
3. Масса адронов на ~90% определяется непертурбативной КХД, что количественно подтверждает идею «вакуумного отклонения».
4. Циклическая космология допустима как bounce-расширение стандартных уравнений Фридмана.

### Не утверждается

1. Это **не** замена общей теории относительности.
2. Это **не** решение проблемы космологической постоянной.
3. `M_Ω` — **не** уникальный схемно-независимый оператор, а бухгалтерский параметр.
4. Темная материя **не** доказанно сводится к скрытым измерениям.
5. Циклическая космология **не** подтверждена наблюдениями.
6. Модель **не** даёт UV-завершения, квантовой гравитации или разрешения сингулярностей.

## 11. Что необходимо для перехода от языка к физике

1. **Новое предсказание**, не воспроизводимое ОТО + известными скалярными ЭТП: сигнатура компактных объектов, лабораторный эффект вакуумно-зависимой инерции.
2. **Нетривиальная модификация геометрии**: эмерджентная метрика, нелокальный отклик вакуума.
3. **Механизм** для проблемы космологической постоянной, а не процедура вычитания.
4. **Микроскопическое происхождение массы**: солитонный, топологический или конденсатный механизм, из которого `M_Ω` следует, а не постулируется.

Наиболее реалистичный путь — адронная КХД: определить вакуумно-структурированную долю инерционной массы через сигма-термы, глюонную энергию и трековую аномалию.

## 13. Заключение

NVG — не альтернативная теория гравитации, а **феноменологическая надстройка** над ОТО, в которой:

- **Ω₀** — вакуумный минимум скалярного поля, а не метафизическое Ничто;
- **масса** — интегральная мера отклонения от вакуума;
- **гравитация** — геометрический отклик метрики на суммарную энергию отклонения;
- **циклы** — bounce-динамика с переносом начальных условий;
- **~91% массы нуклона** — непертурбативная КХД энергия, количественно подтверждающая центральную идею.

Модель внутренне согласована, совместима с существующими наблюдениями и определяет конкретную исследовательскую программу. Но она остаётся интерпретативным языком до тех пор, пока не даст хотя бы одного нового наблюдаемого предсказания, выходящего за рамки стандартной ОТО со скалярным полем.
